e-Fatura Helper

Classifica as tuas faturas pendentes no e-Fatura em segundos — e deixa de perder deduções de IRS que são tuas por direito.

🔒 Corre no teu browser 🙅 Nunca pede a tua password ☁️ Sem servidor — os teus dados não saem da página 👐 Código aberto — vê o que faz

Como funciona

1
Faz login normal no e-Fatura, como sempre fazes.
2
Clica no marcador e-Fatura Helper (instalado abaixo). Ele lê as tuas faturas pendentes e sugere o setor de cada uma com base no teu próprio histórico já classificado.
3
Revês a tabela, corriges o que quiseres, e aplicas só as que aprovas. Só tu decides.

Porque é seguro (e porque devias desconfiar de qualquer site que te peça a password das Finanças)

  • A ferramenta corre dentro do teu browser, na página do e-Fatura onde já estás autenticado. Usa a sessão que já tens.
  • Nunca escreves a password em nada nosso. Não há campo de password. Não a vemos, não a guardamos, não existe.
  • Não há servidor. As tuas faturas e dados fiscais nunca saem da página das Finanças. Nós não recebemos nada.
  • Código 100% aberto. Podes ler exatamente o que faz — no GitHub — ou pedir a alguém que perceba para confirmar.
  • Tu aprovas cada uma. Nada é submetido sem tu marcares e clicares em Aplicar. As sugestões são apenas isso — sugestões.

⚠️ Nunca introduzas o teu NIF e password das Finanças num site que não seja portaldasfinancas.gov.pt. Esta ferramenta nunca to pede — é essa a diferença.

Instalar

Arrasta este botão para a tua barra de favoritos:

📄 e-Fatura Helper

Depois, abre o e-Fatura, faz login, e clica no favorito. (Se o teu browser não deixar arrastar, clica com o botão direito → Adicionar aos favoritos, e cola o endereço do botão.)

A classificação é uma declaração tua à AT. Classifica cada fatura no setor onde a compra realmente aconteceu — o menu deixa escolher qualquer setor, mas a responsabilidade é de quem declara.
